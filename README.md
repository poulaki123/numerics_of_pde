This project solves the Laplace equation on the upper half-disk domain Ω = {(x, y) ∈ ℝ² : x² + y² < 1, y > 0}, with mixed boundary conditions — a Neumann condition on the left part of the boundary and a Dirichlet condition elsewhere. The exact solution is the function g(r, θ) = √r · sin(θ/2), which is harmonic on Ω.
