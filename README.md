This project solves the Laplace equation on the upper half-disk domain Ω = {(x, y) ∈ ℝ² : x² + y² < 1, y > 0}, with mixed boundary conditions — a Neumann condition on the left part of the boundary and a Dirichlet condition elsewhere. The exact solution is the function g(r, θ) = √r · sin(θ/2), which is harmonic on Ω.
The problem involves three parts. First, verifying analytically that g satisfies the Laplace equation ∆g = 0. Second, verifying that the normal derivative ∇g · n vanishes on the Neumann boundary ΓN (the negative x-axis segment). Third, solving the problem numerically and measuring how fast the numerical solution converges to the exact one.
The problem is solved using the Finite Element Method (FEM). The domain is discretized into a triangular mesh, and the solution is approximated using piecewise polynomial basis functions. The experimental convergence rate is computed by comparing the numerical error at different mesh refinement levels.
